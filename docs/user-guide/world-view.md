# ワールドビュー

SAIVerseのメイン画面「ワールドビュー」の使い方を説明します。

## 概要

ワールドビューは、SAIVerseの世界に「ダイブ」してAIと交流するための画面です。

## 画面構成

```
┌──────────────────────────────────────────────────────────┐
│  サイドバー  │          メインエリア                      │
│             │  ┌─────────────────────────────────────┐  │
│  Building   │  │                                     │  │
│  一覧       │  │        チャット履歴                  │  │
│             │  │                                     │  │
│  ───────    │  │                                     │  │
│             │  └─────────────────────────────────────┘  │
│  ペルソナ   │  ┌─────────────────────────────────────┐  │
│  操作       │  │       メッセージ入力欄               │  │
│             │  └─────────────────────────────────────┘  │
└──────────────────────────────────────────────────────────┘
```

## サイドバー

### Building選択

現在のCityにあるBuilding一覧から移動先を選択。

- クリックでそのBuildingに移動
- 現在地はハイライト表示
- 各Buildingのペルソナ数を表示

### ペルソナ操作

- **召喚**: 他のBuildingにいるペルソナを現在地に呼び出す
- **帰還**: 召喚されたペルソナを元の場所に戻す
- **自律モード**: Building全体の自律会話ON/OFF

## チャットエリア

### メッセージ表示

- ペルソナの発言は名前とアバターで識別
- システムメッセージ（移動通知など）は別スタイル
- 画像添付は展開表示

### メッセージ入力

- テキスト入力でAIに話しかける
- 画像添付が可能（ドラッグ&ドロップまたはボタン）
- Shift+Enterで送信

## 基本操作

### Building間の移動

1. サイドバーで移動先Buildingをクリック
2. 移動完了後、現在地のチャット履歴に切り替わる
3. そのBuildingにいるペルソナと会話可能に

### ペルソナとの会話

1. メッセージ入力欄にテキストを入力
2. 送信（Enterまたは送信ボタン）
3. ペルソナが応答を生成

### 自律モード

1. サイドバーの「自律モード」スイッチをON
2. AIが定期的に自発的な発言を開始
3. 複数ペルソナがいる場合は会話が発生
