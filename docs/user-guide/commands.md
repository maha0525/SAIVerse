# コマンド一覧

SAIVerseで使用できるユーザーコマンドを説明します。

## 概要

通常のチャット入力に加えて、特定のプレフィックスを使うことでコマンドを実行できます。

## 利用可能なコマンド

### システムコマンド

| コマンド | 説明 |
|----------|------|
| `/help` | コマンド一覧を表示 |
| `/clear` | チャット履歴をクリア |
| `/status` | 現在の状態を表示 |

### ナビゲーション

| コマンド | 説明 |
|----------|------|
| `/move <building>` | 指定したBuildingに移動 |
| `/summon <persona>` | ペルソナを召喚 |
| `/return <persona>` | ペルソナを帰還させる |

### メモリ操作

| コマンド | 説明 |
|----------|------|
| `/remember <text>` | 記憶に情報を追加 |
| `/recall <query>` | 関連する記憶を検索 |
| `/thread <name>` | スレッド（話題）を切り替え |

### デバッグ

| コマンド | 説明 |
|----------|------|
| `/debug on/off` | デバッグモードの切り替え |
| `/pulse` | 手動でペルソナのパルスを実行 |

## 使用例

### ペルソナを召喚

```
/summon air
```

airというIDのペルソナを現在地に召喚します。

### 特定の話題を思い出す

```
/recall 昨日の会話
```

「昨日の会話」に関連する記憶を検索して表示します。
